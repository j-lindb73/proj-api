# me-api-express

## Project setup

Set up the environment and install dependencies

```
npm install
```

### Start server 

Start server in monitor mode (restart when file changes)

```
npm run start
```


### Documentation
See [Express Website](https://expressjs.com/).


### Reset chatlog

SSH to server.
```
me-api-express/mongodb/node setup.js
```
